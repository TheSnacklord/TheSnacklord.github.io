Copper Wire Project

This project was focused on a thermally insulated copper wire in the ME1600 class.
We worked in a group of five people to complete this project and we used MatPlotLib to solve it. We also had to export the graphs data to another file. I was the primary programmer. Here is the problem description from the presentation slide show. We needed to graph and explicit and implicit heat equation solution for the copper wire, full slideshow available on request This is the explicit solution to the problem and is graphed using MatPlotLib This is the implicit (or approximated) solution to the problem and is also graphed using MatPlotLib Example of input and code which made it easier to test quickly(although it is broken now). The full code is available on my Github under 'ME1600/project2'. There is also the printed data files there as well.
